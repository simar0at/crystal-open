Displays an additional word sketch in another language. The corresponding grammatical relations are aligned. This makes it easy to observe the collocations in both languages. Relations are not aligned if:

  • The same (or comparable) grammatical relation is not included in the word sketch grammar for the second language.
  • The linguistic feature does not exist in both languages.

Statistics
Only monolingual statistics are available to view. The setting is in the view options visibility to display them. No multilingual statistics are available.

Corpora
This tool works with any corpora. A parallel corpus is not required.

Use
Select the corpus in the second language, type the lemma and, if needed, select the part of speech.
