Коли параметр активовано, тезаурус, згенерований із референтного корпусу, буде використано, щоб вивести список подібних слів. Використовуйте його для фільтрування колокатів за семантично пов’язаними словами, наприклад, пес для тварин, манго для тропічних фруктів, рухатися для дієслів руху, щастя для почуттів, жінка для людей тощо.

Більша кількість включатиме більше синонімів у пошук, але вони будуть менше пов’язаними із ключовим словом.
