Тип запиту

Простий

Цей параметр ідентичний простому пошуку на ОСНОВНІЙ вкладці.

Пошук слів, які відповідають пошуковому слову, або %[lemma]% яких (основна форма) відповідає пошуковому слову. Він не чутливий до регістру.

Введіть слово чи фразу. Використовуйте зірочку (*) для будь-якої кількості довільних символів. Використовуйте знак питання (?) для одного довільного символу. Використовуйте вертикальну риску (|), щоб включити більше ніж одне слово або фразу. Використовуйте два дефіси (--) для пошуку слова, що може писатися через дефіс, без дефіса або як два окремих слова.

Лема

Введіть лему, щоб знайти всі %[wordForm|форми слів]% леми. %[regex|Регулярні вирази]% підтримуються.

використання знайти

б.* бути, був, болото, бистрий, більший (слова, лема яких починається на б-)

Фраза

Знаходить багатослівну фразу точнісінько так, як введено. %[regex|Регулярні вирази]% підтримуються. 


Слово

Знаходить конкретну %[wordForm|форму слова]%, точнісінько так, як введено. %[regex|Регулярні вирази]% підтримуються.
