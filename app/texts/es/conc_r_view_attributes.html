La concordancia permite la visualización de %[attribute|atributos]% adicionales. Selecciona los que deberían visualizarse.

Los atributos se numeran al seleccionarlos. Él con numero 1 será visualizado en la fila primaria. Los demás se ponen bajo ella. Por defecto, %[wordForm|forma (word)]% lleva numero 1.

Click filter_list to only display the selected attributes, this is especially useful if the list is too long.

con cada token los atributos seleccionados se visualizarán con cada token

con KWIC only los atributos seleccionados se visualizarán con tokens dentro del %[KWIC]% 

show as tooltips cuando seleccionado, hay que pasar el puntero del ratón sobre el token para visualizar los atributos
