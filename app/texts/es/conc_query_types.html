Tipo de consulta

Simple

This option is identical to the simple search on the BASIC tab.

Finds words which match the search word(s) or whose %[lemma]% (base form) matches the search word. It is not case sensitive.

Introduce una palabra o frase. Usa el asterisco (*) en lugar de una cadena de caracteres sin especificación. Usa el signo de interrogación (?) en lugar de exacto un caracter. Usa la pleca (|) para inlcuir más de una palabra o frase. Usa dos guiones (--) para encontrar una palabra con ortografía variada: una palbra, con guion y como dos palabras.

Ejemplos
introduce para encontrar
m* formas o lemas que empiezan con m-
*ing formas o lemas que terminan con -ing
tener una * idea forma o lema sin especificación
tener * * * idea tres formas o lemas sin especificación
???t formas o lemas de 4 letras que terminan con -t
hispano--romano encontrará hispano-romano o hispanoromano o hispano romano
notar|dar cuenta encontrará tanto notar como dar cuenta

Lema

Type a lemma to find all %[wordForm|word forms]% of the lemma. %[regex|Regular expressions]% are supported. 

Ejemplo
introduce para encontrar
fuimos c.*
c.* color, contará, cuero, quepa (palabras con un lema que empieza con c-)

Forma

Finds a multi-word phrase exactly as typed. %[regex|Regular expressions]% are supported. 


Forma

Finds a specific %[wordForm|word forms]% exactly as typed. %[regex|Regular expressions]% are supported. 

Ejemplo
introduce para encontrar
fuimos fuimos
fuimos fuimos
