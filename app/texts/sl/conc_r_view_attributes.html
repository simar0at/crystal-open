Vsaki pojavnici v konkordanci so lahko dodane male črke, %[lema|lema]%, %[oznaka|oblikoskladenjska oznaka]%, %[lempos]% in drugi atributi, ki so na voljo v korpusu. Izberite tiste, ki naj bodo prikazani v konkordanci.

Atributi so oštevilčeni tako, kot so izbrani. Atribut s številko 1 bo prikazan kot glavni atribut nad preostalimi izbranimi atributi. Privzeto ima %[besedna oblika|beseda]% številko 1.

Click filter_list to only display the selected attributes, this is especially useful if the list is too long.

for each token – the selected attributes will be shown for each token in the concordance

for KWIC only – the selected attributes will only be shown for %[KWIC]% 

show as tooltips – when selected, hover the mouse over the token to display the attributes
